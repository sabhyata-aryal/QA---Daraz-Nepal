# Daraz Nepal QA Project

This repository contains QA work performed for the Daraz Nepal platform, including both Manual Testing and Automation Testing.

---

## Project Overview

The project covers:

- Manual Testing documentation
- Test case design and execution
- Bug reporting and tracking
- Automation testing using Playwright with JavaScript

The objective of this project is to validate functionality, usability, and reliability of the Daraz Nepal web application.

---

## Technologies Used

### Manual Testing
- MS Excel

### Automation Testing
- Playwright
- JavaScript
- Node.js

---

## Automation Framework

The automation framework is built using:

- Playwright with JavaScript
- Page interaction and validation
- Automated browser testing
- Assertions and reporting

---

## Features Tested

The areas tested include:

- User Signup/Login
- Home Page
- Product Search
- Product Details
- Add to Cart
- Shopping Cart
